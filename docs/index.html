Welcome

Azrakel is a moody, mysterious Discord bot crafted for roleplay and server moderation. Developed by Team BarkByte, he handles utility, RP flavor, and admin tasks alike.

Commands

  • /about – Learn about Azrakel
  • /hug – Hug someone
  • /riddle – Get a riddle
  • /joke – Get a joke
  • /howl – Hear Azrakel howl
  • /trail – Forest-themed advice
  • /kick, /ban, /mute – Admin-only moderation tools
  • /warn, /view_warnings, /clear_warnings – Warning system
  • /report – Report a user

Privacy Policy

Azrakel stores only necessary moderation data (like warnings) in local JSON files. No personal data is collected, sold, or shared. All data is server-specific and only accessible to server admins. For concerns, contact us at teambarkbyte@gmail.com.

Terms of Service

By using Azrakel, you agree to use the bot respectfully and within Discord's own terms of service. Abuse or misuse may result in blacklisting or bot removal. Moderators have full discretion.
