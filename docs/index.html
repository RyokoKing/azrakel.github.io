📜 Commands

  • /about – Learn about Azrakel.
  • /hug [user] – Send a warm hug.
  • /riddle – Get a random riddle.
  • /joke – Hear a fun joke.
  • /howl – Azrakel lets out a haunting howl.
  • /trail – Receive a nature-themed tip.
  • /kick [user] [reason] – Kick a user (admin only).
  • /ban [user] [reason] – Ban a user (admin only).
  • /mute [user] [reason] – Mute a user (admin only).
  • /warn [user] [reason] – Issue a warning (mod only).
  • /view_warnings [user] – View user warnings (mod only).
  • /clear_warnings [user] – Clear user warnings (mod only).
  • /report [user] [reason] – Report a user anonymously.

🔐 Privacy Policy

Azrakel does not collect or store personal user data beyond what is required to execute moderation actions within Discord servers. All warning and report data is stored securely in a local environment and is not shared with third parties.
